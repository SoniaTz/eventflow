Η υλοποίηση της εργασίας αυτής έγινε εξολοκλήρου με Vibe Coding, χρησιμοποιώντας αρχικά ένα απλό LLM (ChatGPT), δίνοντάς του την εκφώνηση της άσκησης για να την αναλύσει και να προτείνει τρόπους ώστε η εργασία να είναι πιο λειτουργική. Αρχικά, ζήτησα ένα frontend prompt, το οποίο παρείχα στον Agent της εφαρμογής Figma, ώστε να έχω μια καθαρή και καλαίσθητη εμφάνιση στην προβολή. Έπειτα, ανέβασα αυτό το αρχείο στο IDE Visual Studio Code και, χρησιμοποιώντας δωρεάν agents που λήφθηκαν από τα extensions της εφαρμογής, ζήτησα ξανά από το LLM να μου δημιουργήσει ένα prompt για το backend, ώστε να το προωθήσω στον Agent που εγκατέστησα και να συνδεθεί με το frontend αρχείο. Στη συνέχεια, αφού κατάφερα να δω την εφαρμογή στο localhost, δοκίμαζα τη λειτουργία της, παρατηρούσα λάθη ή έκανα διορθώσεις τόσο στην εμφάνιση όσο και στη λειτουργικότητά της και έδινα ανάλογα prompts, ώστε να διαμορφωθεί όπως θα ήθελα. Αφού ολοκλήρωσα τις δοκιμές και τις διορθώσεις, δημιούργησα, με τη χρήση των agents, ένα αρχείο README, στο οποίο αναγράφονται οι δυνατότητες κάθε ρόλου, ο τρόπος εγκατάστασης των απαραίτητων αδειών, καθώς και οι οδηγίες για την εκτέλεση της εφαρμογής σε localhost, μεταξύ άλλων.Το project έχει ανέβει και στο [GitHub](https://github.com/SoniaTz/spotyourvibe), ώστε να μπορεί οποιοσδήποτε να το ξεκινήσει χρησιμοποιώντας τον κατάλληλο κώδικα. Επίσης η ιστοσελίδα έχει ανέβει στο διαδίκτυο με τον σύνδεσμο [https://www.spotyourvibe.com/](https://www.spotyourvibe.com/)
